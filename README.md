# chatgpt_web 챗봇 웹사이트

## 프로젝트 소개
    
 이 웹사이트는 chatgpt api와 flask를 이용하여 간단한 챗봇 웹사이트를 구현했습니다.

  <br>사용 기술은 python, flask, openapi chatgpt api 입니다.
  
## 실행 방법
chagpt_web 폴더 -> python start.py 실행 <br>
http://localhost:5000, http://127.0.0.1:5000
